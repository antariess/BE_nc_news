Northcoder News API endpoints

GET /api

You are here, viewing this list


GET /api/topics

A list of all topics


GET /api/topics/:topic/articles

A list of all articles for the specified topic


POST /api/topics/:topic/articles

Create a new article for a specified topic


GET /api/articles

A list of all articles regardless of topic


GET /api/articles/:article_id

View an individual article based on ID


GET /api/articles/:article_id/comments

A list of all the comments for a specified article


POST /api/articles/:article_id/comments

Add a new comment for a specified article


PUT /api/articles/:article_id

Upvote or downvote an article


PUT /api/comments/:comment_id

Upvote or downvote a comment


DELETE /api/comments/:comment_id

Delete a comment


GET /api/users/:username

View a user based on username
